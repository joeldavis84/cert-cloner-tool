Server Configuration Examples

Nginx

Upload the .crt and .key to your server and update your server block:

server {
    listen 443 ssl;
    server_name example.local;

    ssl_certificate     /etc/nginx/ssl/server.local.crt;
    ssl_certificate_key /etc/nginx/ssl/server.local.key;

    # Optional: SSL optimization
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
}

Apache httpd2

Ensure mod_ssl is enabled and update your VirtualHost:

<VirtualHost *:443>
    ServerName example.local

    SSLEngine on
    SSLCertificateFile    /etc/apache2/ssl/server.local.crt
    SSLCertificateKeyFile /etc/apache2/ssl/server.local.key
</VirtualHost>
